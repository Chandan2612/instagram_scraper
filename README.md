# Instagram Scraper 📸

A Python-based automation tool built with **Playwright** to scrape data from Instagram profiles. This tool is designed to extract profile information and post data efficiently while handling dynamic content.

## 🚀 Features
* **Profile Scraping:** Extract follower counts, following counts, and bio information.
* **Data Export:** Automatically saves scraped results to `instagram_data.xlsx`.
* **Session Management:** Keeps track of visited profiles in `visited_profiles.txt` to avoid redundant scraping.
* **Headless Mode:** Runs in the background using Playwright's browser engine.

---

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Automation:** [Playwright](https://playwright.dev/python/)
* **Data Handling:** Pandas / Openpyxl (for Excel export)
* **Environment:** Python venv
